<p align="center">
A modern and responsive Library Management System built with <b>Flask</b>, <b>SQLite</b>, and <b>Bootstrap 5</b>. It helps manage books, borrowing, returns, donations, and issued book records through an easy-to-use web interface.
</p>

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/library-management-system.git
```

### Go to Project Folder

```bash
cd library-management-system
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python api/index.py
```

Open your browser:

```
http://127.0.0.1:5000
```
